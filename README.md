# FreeSpeech – TTS API на базе F5-TTS (русский)

Сервис для синтеза речи по образцу голоса.  
Принимает эталонный аудиофайл (WAV) и текст, возвращает сгенерированную речь.

## Возможности

- Русский язык с нормализацией чисел и расстановкой ударений.
- Использование любого голоса (передаётся в запросе).
- Работает на GPU (CUDA) и CPU.
- Упакован в Docker для лёгкого деплоя.
